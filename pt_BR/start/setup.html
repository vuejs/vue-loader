Configurando um projeto

Configurando um Projeto

Usando vue-cli

Recomenda-se o gerador de esqueleto do projeto usando vue-loader com vue-cli:

npm install -g vue-cli
vue init webpack-simple ola-vue
cd ola-vue
npm install
npm run dev # pronto para começar!
